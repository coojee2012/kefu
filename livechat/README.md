## 项目介绍

实时聊天app，兼容web端，基于ionic3，angular4，sip.js


## 环境依赖

nodejs 


### 安装项目依赖
```bash
npm install
```

### 安装ionic、cordova命令行

```bash
npm install -g cordova ionic
```

### 开启服务

```bash
ionic serve
```
浏览器访问：http://localhost:8100，

### build

```bash
ionic build --prod
```
